# fastdecliners

This reposatory contains scripts, data of Phillips et al. 2025:
https://ui.adsabs.harvard.edu/abs/2025arXiv250907093P/abstract

The notebooks  "H0_trippmethod.ipynb" and "H0_colormethod.ipynb" produce
results that are shown in Table 4 of the paper.

These notebooks produce outputs in separate folders within the result folder. Outputs are:
text file containing maginalized valuess of MCMC fitted parameters,
MCMC chains, and corner plots. 


List of distant SNe Ia and list of calibrators for these two methods
are included within respective notebooks. Lists are mathced with
information stored in files in data folder.

A number of SBF calibrators from Garnavich 2023 are added that were not
used in Uddin 2024. These are: 2006mr, 2007on, 2015bp, and 2011iv.
